# Instabot - Instagram Scraping Bot

## Descripción

**Instabot** es una clase de Python diseñada para facilitar el 
scraping de datos de Instagram. Esta herramienta permite a los usuarios extraer información valiosa de 
perfiles, publicaciones y seguidores de Instagram de manera eficiente y automatizada. Con una variedad 
de funciones integradas, Instabot es ideal para investigadores, analistas de datos y desarrolladores 
que buscan obtener información de Instagram para sus proyectos.

## Características

- **Extracción de Seguidores**: Obtén la lista de seguidores de un perfil específico.
- **Recopilación de Publicaciones**: Extrae información sobre las publicaciones de un usuario, incluyendo descripciones, imágenes y comentarios.
- **Filtrado de Usuarios**: Identifica y filtra cuentas basadas en criterios específicos, como nombres de usuario o etiquetas.
- **Manejo de Modales**: Interactúa con pop-ups de Instagram para obtener información adicional, como nombres de usuario en listas de seguidores.
- **Soporte para Etiquetas HTML**: Extrae contenido de etiquetas HTML específicas, como `<span>`, para obtener información detallada.
